# Cellular self-organization orchestrates ferroptosis propagation

## Overview
The customized scripts of the "Cellular self-organization orchestrates ferroptosis propagation", including image analysis, data analysis, scRNA-seq analysis, simulation, and plotting.

Versions of the programming languages are as follows: MATLAB (v. 2024b), ImageJ (1.54p), R (v 4.4.3), and Python (v.3.9.12).

## Folder description
| Folder  | Description |
| :------------- | :------------- |
| IF | Image and statisical analysis for immunofluorescence experiments |
| Laurdan | Image and GP analysis for Laurdan staining |
| PALP | Image and signal processing for PALP staining |
| density | Image analysis for quantifying cell density |
| initiation | Image, statisical, and CNN analysis for initiation characterization |
| orientation | Image and Q-tensor analysis for quantifying cell orientation and topological defects |
| quant_self_org | Image and autocorrelation analysis for waves cultured for different durations of self-organization |
| reaction_diffusion | Mathematicl modeling for reaction-diffusion model |
| scRNA-seq | scRNA-seq analysis, including trajectory inference, RNA velocity, differentially expressed gene analysis, and regulon network |
| wave_analysis | Image and data analysis for wave direction and speed |
